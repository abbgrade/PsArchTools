# New-ArchDataJourney

## SYNOPSIS
Creates a new data journey.

## SYNTAX

```
New-ArchDataJourney [-Title] <String> [<CommonParameters>]
```

## DESCRIPTION
A data journey describes the data layers with models and the data flows in between.
